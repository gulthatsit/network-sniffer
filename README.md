# Network Packet Sniffer
A Python program that captures and analyzes live network traffic.

## Requirements
pip install scapy

## How to Run
sudo python network_sniffer.py

## Features
- Captures TCP, UDP, ICMP packets
- Shows source/destination IPs and ports
- Displays DNS queries and payload previews# network-sniffer
Basic network packet sniffer built with Python and Scapy
